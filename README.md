# 🚗 AI-Based Autonomous Navigation System

## 📌 Project Overview
This project demonstrates an AI-based autonomous navigation system built using Python.  
It simulates how a robot or self-driving vehicle navigates from a start point to a destination while avoiding obstacles.

The system uses the **A* (A-Star) path planning algorithm** to compute the shortest collision-free path in a grid-based environment.

---

## 🎯 Problem Statement
Autonomous systems must navigate efficiently in environments with obstacles.  
This project solves the problem of **finding an optimal path while avoiding collisions**.
